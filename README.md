La función de este proyecto es poder loguear un usuario, y que él pueda crear sus propias tareas y recordatorios.

Tech stack:

* Node
* Express

Demo: https://tarjetitas.herokuapp.com/api/

Clone Project:
```
git clone https://github.com/gnieto11/login-back.git
```

Install dependencies:
```
npm install
```
Run:

```
npm run start
```
